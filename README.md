# RECIPE MASTER 2026

Un projet basé sur une architecture MVC procédurale.

## Installation

- Lancez Mamp/Wamp ou Xamp
- Chargez la db_remplie sur votre serveur mySQL
- Placez le dossier complet dans le dossier racine de votre serveur web (htdocs, wwww, ...)
- Testez via localhost/path_du_dossier.
